### ⚙️ Platform Engineering Philosophy

- 🚀 **High velocity** — self-service platforms so teams ship without waiting on tickets
- ⚡ **Fast, safe deploys** — GitOps-driven CD, automated pipelines end-to-end
- 🔭 **Observability & Reliability** — metrics, logs, and traces as first-class citizens
- 📈 **Scalability by design** — infra and services built to grow without re-architecting
- 🔐 **Security-first (DevSecOps)** — shift-left scanning, policy-as-code, zero-trust by default
